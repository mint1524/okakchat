# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/).

## [0.1.0] - 2026-05-12
### Added
- Flutter multiplatform app (iOS, Android, macOS, Windows, Linux, Web)
- Platform-adaptive UI: Material 3 (Android/Desktop/Web), Cupertino (iOS)
- JWT authentication with auto-refresh and secure token storage
- Email verification flow (register → verify → home)
- Real-time streaming chat via WebSocket
- Markdown rendering with syntax-highlighted code blocks and copy button
- Model selector with multi-model support
- Chat modes: chat and coding
- Conversation history with delete
- Agent mode for desktop: workspace folder, file tree, tool execution with confirmation dialogs
- Admin panel: user management, subscription grants, audit log
- Settings screen with sign out
